# Price Sniffer 管理台 UI Demo

纯静态交互原型，对应产品愿景文档 [PRODUCT.md](../PRODUCT.md)。

## 打开方式

**推荐（避免部分浏览器对本地 `file://` 的限制）：**

```bash
cd docs/ui-demo
python3 -m http.server 8765
```

浏览器访问：http://127.0.0.1:8765/

也可直接双击打开 `index.html`（多数桌面浏览器可用）。

## 功能

- 层级侧栏菜单（可折叠）
- 产品 / 平台 / 市场 / 告警规则 / 抓取任务：增删改查
- 任务状态统计与筛选；任务「立即执行」模拟异步结果
- 任务状态页：历史运行记录、按任务汇总成功率
- 告警面板：触发/确认/恢复/静默状态、筛选与模拟触发
- 详细数据：产品/平台/市场/店铺 + 相对/绝对时间筛选（近一周/月/三月/半年，默认近一个月；手改日期后相对为「-」）；展开 serie 查看价格/销量/销售额趋势
- 价格趋势、按平台/市场聚合趋势（Chart.js + 随机时序）
- 数据持久化在 `localStorage`（模拟 SQLite 元配置）；侧栏可「重置演示数据」

正式架构为 Go + SQLite 元配置 + TDengine 时序 + H5，本 Demo 无后端。
