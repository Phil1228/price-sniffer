# Price Sniffer 管理台 UI Demo

纯静态交互原型，对应产品愿景文档 [docs/PRODUCT.md](../docs/PRODUCT.md)。

## 打开方式

```bash
cd ui-demo
python3 -m http.server 8765
```

浏览器访问：http://127.0.0.1:8765/

也可直接双击打开 `index.html`。

## 功能

- 层级侧栏菜单（可折叠）
- 产品（含分类）/ 平台 / 市场 / 告警规则 / 抓取任务：增删改查
- 任务状态：历史运行、成功率
- 告警面板：触发/确认/恢复/静默
- **分类销量价格汇总**：按产品分类聚合均价、销量、销售额
- **详细数据**：产品/平台/市场/店铺 + 时间筛选；关联告警规则显示标识；展开 serie 看趋势
- 价格趋势、分类聚合趋势（Chart.js）
- `localStorage` 持久化；侧栏可重置演示数据
